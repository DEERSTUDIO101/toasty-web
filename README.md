# 🔥 Toasty Hub

Toasty Hub Roblox Script Hub Website

## Setup

1. Put your banner image as `images/toastie-banner.png`
2. Put your mascot image as `images/toasties.png`

## Deploy to Netlify

1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → New site from Git
3. Pick your GitHub repo
4. Build command: *(leave empty)*
5. Publish directory: `.`
6. Deploy!

The `netlify.toml` is already included.
